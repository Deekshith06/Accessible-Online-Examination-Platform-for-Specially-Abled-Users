♿ Core Accessibility

Features Built for Every User

Every feature is designed with inclusivity as the foundation, not an afterthought.

👁️

Visual Impairment Support

  • 🔊 Text-to-Speech for all questions
  • ⌨️ Full keyboard navigation
  • 🎨 High contrast mode
  • 🔍 Adjustable font sizes (up to 3x)
  • 🖥️ Screen reader compatible (ARIA)
🧏

Hearing Impairment Support

  • 📄 All instructions in text format
  • 🎥 Optional sign-language video guides
  • 📢 Visual alerts instead of audio
  • 💬 Caption support for all media
  • 🔔 Visual notification system
🧑‍🦽

Motor Disability Support

  • 🗣️ Voice command navigation
  • 🎙️ Speech-to-Text for answers
  • ⌨️ Tab-based simple navigation
  • 🖱️ Large click targets
  • ⏸️ Pause exam option
